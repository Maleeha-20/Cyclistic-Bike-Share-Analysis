## Project Objective

This project analyzes bike-share usage patterns to understand behavioral differences between member and casual riders, with a focus on identifying opportunities for membership growth.

## Dataset

Approx. 4.29 million ride records

## Tools & Technologies

PowerPoint (Stakeholder Presentation)

## Key Insights

Casual riders exhibit longer ride durations, suggesting leisure-oriented usage.
Members drive the majority of platform usage.
Electric bikes demonstrate higher ride volumes across segments.
Ride demand follows clear seasonal patterns.


## Strategic Recommendations

Implement targeted casual-to-member conversion strategies.
Align fleet allocation with electric bike demand.
Deploy seasonally adaptive pricing and promotional initiatives.
